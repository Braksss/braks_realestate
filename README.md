# Market Pulse - Architecture SaaS Complète

Ce projet est structuré en monorepo et contient 3 services indépendants :

- **/webapp**: L'application frontend en Next.js, visible par les utilisateurs.
- **/cms**: Le backend de contenu (Headless CMS) pour gérer les guides et autres textes.
- **/scraper**: Les scripts Python pour la collecte de données.
