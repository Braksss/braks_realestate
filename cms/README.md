# CMS (Backend de Contenu)

Ce dossier contiendra notre Headless CMS, basé sur Strapi.

## Installation (à faire une seule fois)
1.  Assurez-vous d'avoir Node.js (version 18 ou supérieure) installé.
2.  Ouvrez un terminal DANS CE DOSSIER (/cms) et lancez la commande suivante :
    
px create-strapi-app@latest . --quickstart
3.  Suivez les instructions pour créer votre compte administrateur.
